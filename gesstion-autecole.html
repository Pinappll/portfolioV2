Information du Projet

  • Catégorie: Application Desktop Java
  • MODELE: MVC
  • Langages: Java, Interface Swing, Sql
  • Outil: IntelliJ, mySql, Mamp
  • STATUT: En cours

Contexte

Gestion-Autoécole est une application Java de gestion complète pour les auto-écoles. Elle permet de gérer efficacement les membres, les moniteurs et les véhicules, tout en permettant une planification simple et efficace des heures de conduite. Cette application est un outil indispensable pour les auto-écoles souhaitant gérer efficacement leur activité.

Objectifs de l'appli

  • - Gestion-Autoécole est une application Java de gestion pour les auto-écoles. Cette application a été conçue pour aider les auto-écoles à gérer les membres, les moniteurs et les véhicules, ainsi que les heures de conduite attribuées à chaque membre.
  • - Gestion des membres : L'application doit permettre de gérer les membres de l'auto-école, y compris leurs informations personnelles, leur niveau de conduite et leur progression.
  • - Gestion des moniteurs : L'application doit permettre de gérer les moniteurs de l'auto-école, avec des informations telles que leur disponibilité, leurs heures de travail et leur spécialisation.
  • - Gestion des véhicules : L'application doit permettre de gérer les véhicules de l'auto-école, y compris leur disponibilité, leur type et leur état.
  • - Planification des heures de conduite : L'application doit permettre aux administrateurs de planifier des heures de conduite pour chaque membre en fonction de leur niveau de conduite, et d'attribuer ces heures à un moniteur spécifique pour un véhicule particulier.

Design et ergonomie de l'appli

  • - Interface utilisateur : L'application doit offrir une interface utilisateur simple et conviviale, avec une navigation facile et une présentation claire des informations.
  • - Navigation claire et intuitive pour faciliter l'accès aux différentes rubriques de l'application
